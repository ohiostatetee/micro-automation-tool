# 🟨 Micro Automation Tool

A lightweight Python automation tool for running small, repeatable tasks.

✨ Features
- Task registry with named tasks
- CLI to run tasks by name
- Simple logging to console and file
- Easy to extend with new tasks

🚀 Use Case
Designed for repetitive evaluation tasks, automation pipelines, and micro‑task operations.

📦 Installation
pip install -r requirements.txt

▶️ Usage
python automation_tool.py list
python automation_tool.py run hello
python automation_tool.py run health_check
